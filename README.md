# BPF-Trace-Utility-Tool
Find Linux kernel insights about kprobe and tracepoints using bpftrace 

### Example of usage
![example](/example.gif)

# Usage
Requirements: bpftrace

1. Install bpftrace
```bash
sudo apt update
sudo apt install bpftrace 
```
2. start the program

```bash
./bpf-tracer-utility.sh
```
